# sales-Dashboard
• Developed an interactive Power BI dashboard to analyze sales performance, profit trends, and product-wise contribution across regions. • Created DAX measures to calculate KPIs such as Total Sales, Profit, Average Margin %, and Month-over-Month Growth %. 
🟣 Sales Analytics Dashboard (2025) – Power BI
📊 Project Overview

This project focuses on analyzing and visualizing sales performance metrics using Power BI.
The dashboard provides a comprehensive view of key indicators such as Total Sales, Profit, Quantity, and Growth Rates, helping businesses make data-driven decisions and track monthly sales performance efficiently.

🎯 Objectives

Analyze overall sales and profit trends across months and regions.

Identify top-performing products and high-revenue categories.

Measure monthly growth rates and average profit margins.

Provide business users with interactive filters to explore sales insights dynamically.

⚙️ Key Features

KPI Cards: Display Total Sales, Total Profit, Quantity Sold, Average Margin %, and Month Growth %.

Trend Analysis: Visualize sales over time using line and area charts.

Regional Insights: Compare total sales by region (East, West, North, South).

Product Insights: View sales contribution by product name and category (Grocery, Electronics, Clothing, Furniture).

Filters & Slicers: Dynamic selection by Month and Product for deep dive analysis.

Modern Theme: Designed with a purple–black gradient theme for a clean and futuristic look.

🧠 Technical Highlights

Built using Power BI Desktop with DAX measures for KPIs and growth rate calculations.

Data pre-processed and cleaned using Power Query Editor.

Interactive visuals implemented with slicers and cross-filter interactions for better exploration.

Integrated Year-over-Year (YoY) and Month-over-Month (MoM) growth metrics using DAX formulas.
